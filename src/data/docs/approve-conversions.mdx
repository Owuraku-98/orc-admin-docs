# Steps to Approve Conversions
Go to Pending Approvals
On the dashboard, select “Amendments & Filings Pending Approval”.

![2000X1238](/approve-conversions.png)

This section contains all transactions that require review — including conversions.
Locate the Conversion Transaction

![2000X1238](/approve-conversions1.png)

Use the filter options to quickly find the conversion request:
Entity Type – Choose the relevant business type (e.g., Private Company Limited by Shares).
Service Type – Select Conversions.
Payment Type – Choose All Payment or Paid to see completed submissions.
Date Range or Search – Enter the entity name or transaction number to narrow the results.

![2000X1238](/approve-conversions2.png)

Open the Conversion Details
Click on the Transaction Number (e.g., 20251024000008987).
