# Overview

A registered entity may be assigned to one user but require approval by another. The Transaction Timeline feature helps the user locate, retrieve, and assign such entities.

## Steps

### Transaction Timeline tab

- Click the **Transaction Timeline** tab on the dashboard's horizontal navbar after the user logs in.

![Approve Registrations|2000X1238](/approvereg1.png)

### Filter

- Use the filter to quickly find the target entity. The user can choose different filters as shown in the image below. This example filters entities by **name**.

![Approve Registrations|2000X1238](/approvereg2.png)

### Retrieve

- After locating the target entity, click the three vertical dots under the Actions column.

![Approve Registrations|2000X1238](/approvereg3.png)

- Select **Retrieve**.

![Approve Registrations|2000X1238](/approvereg4.png)

- Click **Submit**.

![Approve Registrations|2000X1238](/approvereg5.png)

- The entity will then be assigned to the current user, as shown below.

![Approve Registrations|2000X1238](/approvereg6.png)
