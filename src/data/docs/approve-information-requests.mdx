# Steps to Approve Information Request

Go to the Pending Section
On the main dashboard, locate and click “Amendments & Filings Pending Approval” or “Information Requests” (depending on how your portal labels it).
This section lists all transactions awaiting your action — including filings and information requests.

![2000X1238](/approve-information-requests.png)

##  Locate the Information Request

Use the filters at the top to narrow your search:
Entity Type – Select the type of business or entity.
Service Type – Choose Information Request if available.
Payment Type / Date Range / Search – Use these to find a specific request quickly.
Look through the list until you find the transaction number associated with the request.

![2000X1238](/approve-information-requests1.png)

### Open the Request Details

Click the transaction number (e.g., 20251024000008987).
This will open the Information Request Details page, showing:
The entity name (e.g., Hisense)
The registration number
The requested information and attached documents.

![2000X1238](/approve-information-requests2.png)

#### Review the Request
Go through the submitted or corrected information carefully.
Verify that all requested details or documents have been properly provided.
Check compliance and consistency with registration requirements.

![2000X1238](/approve-information-requests3.png)

Take Action: Approve or Query
If the information provided meets all requirements:
Click Approve to confirm that the request has been satisfied.
If the submission is incomplete or incorrect:
Click Query, then enter your comments or the missing requirements.
The applicant will receive a notification to update or resubmit.

![2000X1238](/approve-information-requests4.png)

Confirm Approval
Once approved, the system updates the transaction status from Pending or Queried to Approved.
The transaction disappears from the pending tray or moves to the approved list.
Return to Main Page
Click Back or Amendments & Filings Pending Approval to return to the main list and proceed with other approvals.


![2000X1238](/approve-information-requests5.png)
