# Steps to Approve Document Request

## Navigate to Pending Approvals

![2000X1238](/approve-document-request.png)

###	From the dashboard, select “Amendments & Filings Pending Approval” or “Document Requests”.

![2000X1238](/approve-document-request1.png)

### This section shows all requests currently awaiting your review and approval.

![2000X1238](/approve-document-request2.png)

### Locate the Document Request
Use the filters at the top of the page to find the document request:
Entity Type: Select the relevant business type.
Service Type: Choose Document Request if listed.
Payment Type, Date Range, or Search Bar: Narrow down the results by date, payment type, or entity name.
Review the list until you find the transaction you want to process

![2000X1238](/approve-document-request3.png)

### Open the Request Details
•	Click on the Transaction Number (e.g., 20251031000009327).
•	This opens the detailed request page showing:
The Entity Name (e.g., JEEBIS JUICE)
The Registration Number
The Type of Document Requested (e.g., Basic Entity Information, Full Entity information, Directors of Company, Auditor of Company)
Any uploaded supporting files or payment confirmation.

![2000X1238](/approve-document-request4.png)

### Review the Request Information
Check that:
The request is valid and complete.
The required documents and fees have been submitted.
The request matches the entity’s records in the system.

Take Action on the Request
After reviewing, scroll down or look for the action buttons.
Choose one of the following:
✅ Approve – if the request and documentation meet all requirements.
❓ Query – if additional information or corrections are needed. Enter your comments and submit for the applicant to respond.
Confirm the Approval

![2000X1238](/approve-document-request5.png)

Once approved:
The status of the request changes from Pending to Approved.

![2000X1238](/approve-document-request6.png)

The applicant will be notified that their requested document (e.g., certificate or copy) is ready for download or collection.
Return to the Main List
Click back or go to Amendments & Filings Pending Approval to review other document requests.
You can also use Reset Filters to refresh the list.
