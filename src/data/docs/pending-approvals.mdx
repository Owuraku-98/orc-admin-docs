# Overview

This section guides the user through the approval process. Pending approvals can be accessed when a registered entity has been assigned to the current user or after a successful retrieval.

## Steps

### Pending Approvals tab

- After retrieval is complete, or when the user logs in to the dashboard, click the **My Pending Approvals** tab on the horizontal navbar.

![Approve Registrations|2000X1238](/approvereg7.png)

- Select the **Registrations Approval** option.

![Approve Registrations|2000X1238](/approvereg8.png)

- Locate the entity to be approved and click the transaction ID under the **Transaction** column.

![Approve Registrations|2000X1238](/approvereg9.png)

### Review and approval

- After clicking the transaction ID, the user will be taken to a review page.
- Review the details and confirm they are accurate.
- Scroll to the bottom of the page and click the **Approve** button.

![Approve Registrations|2000X1238](/approvereg10.png)

- A warning popup will appear. Read it carefully, then click **Approve** to confirm.

![Approve Registrations|2000X1238](/approvereg11.png)

The entity will be approved successfully after completing the steps above.
